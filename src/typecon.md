# Type conversion

```python
a = int(11) # a will be 11
b = int(2.812) # b will be 2
c = int("34") # c will be 34
```

Output

```python
11
2
34
```



```python
f = float(8.5) # f will be 8.5
d = float("7") # d will be 7.0
```

Output

```python
8.5
7.0
```

k = str("string1") # k will be 'string1'
